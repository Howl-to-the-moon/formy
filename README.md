# formy
formy the form is a form of practicing form making

Formy is a practice page for the Odin Project's
'Sign-up Form' project. 

Skills involved:
> Managing Github branches

> Using HTML and CSS to make a minimalistic sign up page

> Using HTML to make a working (and accessible) form

> Using CSS Pseudo-classes to make said form more interactable
